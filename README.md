# Chat manager
Chat manager is a powerful plugin for making what you want with chat. It is a port of Java Chat Manager.
# Features:
* Local and global chat system
* Good Configuration
* Banwords system
* Simple chat-bot
* Prefix system
* Anti-caps(maybe work incorrectly)
* Simple mute system
